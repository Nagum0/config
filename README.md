# NEW NEOVIM CONFIG!!!

- Package manager: lazy.nvim (https://github.com/folke/lazy.nvim)
- Statusline: Lualine (https://github.com/nvim-lualine/lualine.nvim)
- Plenary: Lua async functionality (I don't use it directly) (https://github.com/nvim-lua/plenary.nvim)
- Harpoon: Used for quickly jumping between files (https://github.com/ThePrimeagen/harpoon)
- Comment: Used for commenting (https://github.com/numToStr/Comment.nvim)
- LSP: Mason (https://github.com/williamboman/mason.nvim); I will also mention this VonHeikemen's lsp-zero because I use it for the main lsp setup (https://github.com/VonHeikemen/lsp-zero.nvim)
- Treesitter: Used for syntax highlighting improvements (https://github.com/nvim-treesitter/nvim-treesitter)
- File search: Telescope (https://github.com/nvim-telescope/telescope.nvim)
- Theme: Gruvbox (https://github.com/ellisonleao/gruvbox.nvim)
